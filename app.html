7-zip命令行的使用，先将7-zip加到系统的环境变量里

解压文件：7z x{fileName}-o{outputDirectory}

#解压C盘下的1.zip到C盘下的test目录里（注意：o后面没有空格）7z x C:\1.zip-oC:\test#解压当前目录下的1.zip到当前目录下的test目录里7z x 1.zip-otest

添加文件：7z a

#添加当前目录下所有的文件到1.7z的压缩包（没有则创建）7z a 1.7z#添加当前目录下test目录里的所有文件到1.7z压缩包里7z a 1.7z test/

删除文件

#删除1.zip压缩包中的所有扩展名为bak的文件：7z d 1.zip*.bak-r

命令行里输入7z.exe可以查看其它可用操作：

  • a：将文件添加到压缩档案中
  • b：测试压缩或解压算法执行时的CPU占用
  • d：从压缩档案中删除文件
  • e：将压缩档案中的所有文件解压到指定路径，所有文件将输出到同一个目录中
  • h：计算文件的哈希值
  • i：显示有关支持格式的信息
  • l：列出压缩档案的内容
  • rn：重命名压缩档案中的文件
  • t：测试压缩档案的完整性
  • u：更新要进入压缩档案中的文件
  • x：将压缩档案中的所有文件解压到指定路径，并包含所有文件的完整路径

    字符作用：示例

    * 通配符：*.pdf（任何名后缀为pdf文件）

    space 与：01 ok（包含01与ok）

    | 或：123|456（包含123或456的文件）

    dm:日期：*jpg dm:2016/10/10-:2016/10/20（2012年10月10日到20日的.jpg文件）

    X:\XX 路径：D:\folder 123（D盘folder文件夹下包含123的文件）

    size:大小：*.zip size:>300MB（所有大于300MB的.zip文件）

    ? 通配符：???.txt（xxx.txt文件（和*的区别是一个?号代替1个字母）

    . 任意单字符：nj.hen（nj任一字符hen，结果njshen）

    ^ 头部：^game（以game开头的）

    $尾部：game$（以game结尾的）

    [] 包括字符：[abc]（a或b或c）

    [^] 不包括字符：[^abc]（不包括a或b或c）

    [-] 任意字符：[a-z]（从a到z之间的任何一个字符）

    [--] 任意字符：[a-zA-Z]（从a到z,及从A到Z之间的任何一个字符）

    常用命令

    ffmpeg -i file.mp4 -hide_banner # 查看视频编码和比特率，也可以接 http://...mp4 查看网络视频
    ffmpeg -i input.mp4 -c:v libx264 output.mp4 # 转成 H.264/5 编码
    ffmpeg -i input.mp4 -c copy output.webm # 编码不变的转换为 webm 格式
    ffmpeg -i input.mp4 -minrate 964K -maxrate 3856K -bufsize 2000K output.mp4 # 调整码率最小为964K，最大为3856K，缓冲区大小为2000K
    ffmpeg -i input.mp4 -vf scale=720:-1 output.mp4 # 改变分辨率为720P
    ffmpeg -i input.mp4 -vn -c:a copy output.aac # 提取视频中的音频
    ffmpeg -i input.aac -i input.mp4 output.mp4 # 合并音频视频
    ffmpeg -i input.m4a -i input.mp4 -vcodec copy -acodec copy output.mp4 # 合并音频视频
    ffmpeg -y -i input.mp4 -ss 00:01:24 -t 00:00:01 output_%3d.jpg # 从指定时间开始连续对1秒钟的视频进行截图
    ffmpeg -ss 01:23:45 -i input -vframes 1 -q:v 2 output.jpg # vframes 1 指定只截取一帧（即一张图），-q:v 2 表示图片质量（1-5，1为质量最高）
    ffmpeg -i input.mkv -vf subtitles=text.ass 01_output.mkv # 添加 ass 字幕
    ffmpeg -i input.mp4 -i file.srt -c:s mov_text -c:v copy -c:a copy output.mp4 # 以文本形式添加字幕

    剪切片段，输出为一个新视频。可以指定开始时间（start）和持续时间（duration），也可以指定结束时间（end）
    ffmpeg -ss 00:01:50 -i [input] -t 10.5 -c copy [output]
    ffmpeg -ss 2.5 -i [input] -to 10 -c copy [output]
    为音频添加封面，以视频格式上传
    ffmpeg -loop 1 -i cover.jpg -i input.mp3 -c:v libx264 -c:a aac -b:a 192k -shortest output.mp4 # -loop 1 表示图片无限循 cover.jpg，input.mp3，-shortest 表示音频文件结束，输出视频就结束

    --sub-lang：指定字幕语言，多个语言用逗号分隔
    --convert-subs：转换字幕格式，srt 较通用；若不转，可能是 .vtt 的；如果有 ass 字幕可下载，则无须加此项

    使用方法

    下载后将 tinyfilemanager.php 文件（可修改为 index.php ）以及 translation.json 文件（多国语言）上传到服务器目录里即可直接使用。

    上传之前可以做如下配置：

    修改账户密码：默认有两个用户，admin 和 user，先访问 php 密码生成，然后替换 tinyfilemanager.php 中如下代码部分： 
    // Users: array('Username' => 'Password', ...)
$auth_users = array(
    'admin' => '替换这里', //admin（这是默认密码）
    'user' => '替换这里', //12345（这是默认密码）
);
    如果不想使用账户密码可以修改此部分：$use_auth = true;（改为 false 为禁用

    设定根目录：默认会访问服务器的根目录，我这里指定了 disk 目录，只想允许访问这个目录，那么可以修改如下部分：
    // Root path for file manager
// use absolute path of directory...
$root_path = $_SERVER['DOCUMENT_ROOT'].'/disk';（即 加入 .'/disk' ）
    隐藏文件： 
    
$exclude_items = array(
    'abc',（隐藏 abc 的文件夹）
    'a.mp4',（隐藏 a.mp4 文件）
    '*.php' （隐藏 php 扩展名的文件）
);
