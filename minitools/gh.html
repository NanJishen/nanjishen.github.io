加速 GitHub 文件下载

GitHub 文件链接带不带协议头都可以，支持 release、archive 以及文件，右键复制出来的链接都符合标准（注：不支持项目文件夹）

分支源码：https://github.com/xxx/project/archive/xxx.zip

release 源码：https://github.com/xxx/project/archive/xxx.tar.gz

release 文件：https://github.com/xxx/project/releases/download/v1.0/xxx.zip

分支文件：https://github.com/xxx/project/blob/master/filename
